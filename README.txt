# Web de Rubén Muñoz — GitHub Pages (gratis)

## Cómo publicar
1. Crea un repositorio en GitHub (por ejemplo, `rubenmuñozdocente`).
2. Sube todo el contenido de esta carpeta a la raíz del repo.
3. Ve a Settings ▸ Pages ▸ Source ▸ Deploy from a branch ▸ `main` / `/ (root)`.
4. Pulsa Save y espera a que aparezca la URL pública.

## Después de publicar
- Edita en `index.html` la etiqueta `<link rel="canonical" ...>` y en `robots.txt` y `sitemap.xml` reemplaza `TU_USUARIO` y `TU_REPO` por los reales.
- Para refrescar la caché de imágenes, usa Ctrl+F5.

## Estructura
- `index.html` — página principal
- `assets/` — imágenes y favicon
- `robots.txt`, `sitemap.xml` — SEO
